# Nathan Lin — Portfolio

Personal portfolio site built with Astro, React, and Tailwind CSS. A cosmic-themed single-page site showcasing projects, experience, and skills.

## Stack

- [Astro](https://astro.build) 6 — static site framework
- [React](https://react.dev) 19 — interactive components
- [Tailwind CSS](https://tailwindcss.com) 4 — styling
- TypeScript

## Project Structure

```text
/
├── public/
├── src/
│   ├── components/    # React components (Hero, About, Projects, etc.)
│   ├── layouts/       # Astro layouts
│   ├── pages/         # Astro routes
│   └── styles/        # Global CSS + theme tokens
└── package.json
```

## Commands

Run from the project root:

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Install dependencies                         |
| `npm run dev`     | Start local dev server at `localhost:4321`   |

Requires Node.js `>=22.12.0`.
